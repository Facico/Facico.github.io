简单图像处理

平滑

滤波

大多数的滤波都有一定的去噪作用

  • 均值滤波
    • 就是在小方框内取均值
  • 方框滤波：
    • 就是在均值滤波之后做归一化
  • 高斯滤波：
    • 就是直接用高斯分布公式来处理，可以处理掉异常值
  • 中值滤波
    • 类似均值滤波，就是用中值代替，可能类似的也有众数滤波

形态学处理

==假设现在的二值图像是黑底白字的==

腐蚀操作

  • 就是方框内取最大值，对于二值图像有黑到白的腐蚀的作用
    • 可以去毛刺

膨胀操作

  • 就是方框内取最小值，对于二值图像有白到黑的膨胀的作用
    • 可以填充图像内部的空洞

开运算与闭运算

  • 运用腐蚀与膨胀来作用

开运算

  • 先腐蚀后膨胀
    • 先取最小值再取最大值
    • 在黑色里面的白色缩小了，达到了去除孤立点的作用，去毛刺等

闭运算

  • 先膨胀再腐蚀
    • 先取最大值再取最小值
    • 在黑色的里面的白色被扩张了，达到了将两个细微连接的图像平起来的作用(把刺扩大)

​

梯度运算

膨胀-腐蚀运算

  • 这里的梯度=膨胀-腐蚀
    • 很显然会得到边缘

礼帽和黑帽

  • 礼帽 = 原始输入 - 开运算结果
    • 得到刺
  • 黑帽 = 闭运算结果 - 原始输入
    • 得到刺的边缘
